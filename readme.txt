List the upcoming posts on your WordPress as a widget.

== Description ==

This plugin just provides a list of the upcoming posts on your WordPress as a widget.

We will set post 'Schedule' in WordPress for the maintenance of a blog update frequency. If the site shows these pre-release articles, will attract readers continue to focus on your blog?

== Installation ==

1. Upload the whole `wp-upcoming-posts-widget` directory  to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget in 'Widgets' Menu under Appearance

== Frequently Asked Questions ==

= I can't see any output in the sidebar after I active the widget =

Just schedule a post. It will not appear unless there is something.
